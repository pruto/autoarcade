
### 食用方法

- 切换1:1像素的画面16:9，窗口大小为1067x600
- 亮度、对比度、伽马值、饱和度调0
- 进入街机拼图页面，点击【192片】和【第一个拼图】
- 开跑
- 切换窗口后autoarcade会自动结束

### 问题和原因

- 调亮度等值是为了确保图片文件和拼图取色一致，打了补丁导致图片不一样的也可以选择截图替换掉refer.png
- 有些运行失败情况可以通过管理员身份运行解决
- 有安全软件拦截导致没法自动切换到前台的，尝试加了一段修复代码
